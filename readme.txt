== Description ==

You can write a simple post in frontend without login to backend.
You can easily to post something like Twitter. Write Sideblog is also good idea.

== Screenshots ==

1. When you login your blog as administrator. this form will show out. And others that do not have "edot post" competence will not see this thing.

2. You just need to add this widget. No need to set anything.

== Installation ==

1. Upload this plugin to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Done!
